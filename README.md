📝 ToDo App (Flutter)

A simple ToDo list app built with Flutter.
You can add, edit, mark as done, and delete tasks.

🚀 Features

Add tasks
Edit tasks
Mark tasks as completed
Delete tasks with confirmation

▶️ How to Run

1. Clone the repo
git clone <your-repo-url>
cd <repo-folder>

2. Get dependencies
flutter pub get

3. Run the app
flutter run
